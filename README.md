**Fonctions :**

-   Allume / Ouvre / Eteint / Ferme + ( module + pièce ).

**EX :**

- Allume la lumère du salon
- Eteint la cafetière de la cuisine
- Ferme les volets de la chambre.

**Configuration :**

Dans le fichier fibaro.prop

      "ip": "192.168.X.X",
      "port": 80,
      "user": "USER",
      "password": "PASSWORD",
